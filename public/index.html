Flu data API

  • Get the latest week's flu data with this request:
        http://fluharvest.mybluemix.net/latest
  • Or get the report for a single state by providing it's abbreviation in the request:
        http://fluharvest.mybluemix.net/latest?state=CA
  • You can also get this data in a map-ready format by requesting the report information be attached to state geographies:
        http://fluharvest.mybluemix.net/latest?geo=true
